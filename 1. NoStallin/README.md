# NoStallin App

NoStatllin app allows the users add and read information about a men's restroom having a door or not. This crowd sourced data is then used to compute number of votes whether a restroom has a door or not.

Features 
===================

1. App gives suggestion about the nearby places.
2. User can select the place, to check the information.
3. If no information is available/ if user wants to add new information, they can add/provide their feedback.
4. They can check the information about a particular place based on the feedback of other users.
